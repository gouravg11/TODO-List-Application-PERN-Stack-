# Time Tracker Project

This is a time tracking app built using the PERN stack (PostgreSQL, Express, React, Node.js). The app helps you keep track of your tasks, add new tasks, edit them, and delete.

## Features

- **Add Task**: You can add new tasks to your project.
- **Edit Task**: You can update details of a task.
- **Delete Task**: You can delete tasks you no longer need.

## Tech Stack

- **PostgreSQL**: Database to store task and time information.
- **Express**: Backend framework to handle requests.
- **React**: Frontend library to create the user interface.
- **Node.js**: JavaScript environment to run the server.

## Installation

1. **Download the project**: Get the project files on your local computer.
2. **Set up the Backend**: Install the backend dependencies and set up your database using a `.env` file for your database credentials.
3. **Set up the Frontend**: Install the frontend dependencies and start the React app.

## How to Use

1. Open the app in your browser.
2. Add tasks and track time for each task.
3. Edit or delete tasks when needed.
